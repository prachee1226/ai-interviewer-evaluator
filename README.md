# AI Interviewer Evaluator

A simple Python-based AI Interview Evaluator that calculates candidate scores using weighted evaluation metrics.

## Features
- Accuracy evaluation
- Depth evaluation
- Clarity evaluation
- Weighted final score calculation

## Technologies Used
- Python

## Formula
Final Score = (Accuracy × 0.5) + (Depth × 0.3) + (Clarity × 0.2)

## How to Run

```bash
python app.py
